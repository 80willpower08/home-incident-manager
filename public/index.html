Automation Mode

Toggle each incident type between Recommend (the AI suggests, you approve) and Auto (the AI acts on high-confidence decisions).

Notification Targets

Set the Home Assistant notify.X service name for each household member — this is where their resolution/denial notifications go. Find service names in HA Developer Tools → Services (search for notify). Admin events use the HA_NOTIFY_ADMIN env var.

Users

Manage local accounts (only shown when AUTH_MODE is local). Each user can submit incidents; admins can approve, deny, and manage settings.

Categories

Incident types available in the Submit form. Add new ones for anything your setup needs (UPS, NAS, media requests, home automation, etc.). Deactivate instead of deleting if incidents already reference the category.

Policies

Markdown policy documents that guide the AI's decision-making. The AI receives enabled policies for the relevant category as part of its evaluation prompt.
